# Molecules - Vibe Kanban

Molecules combine atoms into reusable UI patterns.

## Input groups
- Label + input + helper text
- Label + select + helper text
- Toggle or switch with inline label and tooltip
- Search bar: leading icon + input + optional clear action
- URL editor: read-only field that switches into an editable input

## Card header patterns
- Title + actions row (ellipsis menu, icon buttons)
- Title + metadata row (date, status)
- Breadcrumb header with optional close action

## List and row items
- Project list card: title, metadata, actions menu
- Todo list item: status icon + text, optional strikethrough for cancelled
- Table row: compact cells with click-to-open behavior

## Menu items
- Icon + label menu item
- Destructive menu item with warning color
- Grouped sections with separators and section labels

## Avatar + name combinations
- Small circular avatar with initials or image
- Inline avatar preceding title
- Tooltip label on avatar for full name/assignment

## Action groups
- Toggle group for view modes (preview, diffs)
- Icon button cluster for toolbar actions (refresh, copy, open, stop)
- Inline action cluster on cards (status icons + actions menu)
