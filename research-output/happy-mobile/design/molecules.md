# Molecules

Simple component combinations used repeatedly.

## List row (standard)
- Icon (left) + title + optional subtitle + optional trailing detail + chevron
- Variants: with loading spinner, destructive state, selected state
- Uses divider and pressed feedback

## Grouped list section
- Uppercase section label
- Rounded card container for rows
- Optional footer helper text

## Tab item
- Icon + label
- Optional badge (count) or dot (unread)

## Avatar row
- Avatar + primary name + secondary metadata
- Optional status dot + status text

## Message bubble
- Container with markdown text
- Variants: user bubble (filled) vs agent text (transparent surface)

## Tool card header
- Icon + tool title + status (spinner or check)
- Optional subtitle / elapsed time

## Input group
- Label + input field + helper text
- Helper text used for warnings or hints

## Action button group
- Stacked primary + secondary actions
- Used in confirmation and onboarding screens

## Permission selector row
- Icon + label inside a chip / pill
- Tap to cycle modes (visual state changes)

## Badge + icon combo
- Small badge attached to icon for unread/attention
- Typically positioned top-right of icon
