# Organisms

Complex UI sections composed of multiple molecules.

## Global navigation
- Phone: top header + bottom tab bar
- Tablet: persistent sidebar + content panel

## Sidebar navigation (tablet)
- Header with logo, connection status, and quick actions
- Navigation shortcuts (inbox, sessions, settings)
- Voice assistant status bar under header

## Sessions list
- Sectioned by “active”, “projects”, or recent groups
- Rows include avatar, title, subtitle, status dot, and swipe actions
- Inline banner for updates or announcements

## Chat / session view
- Header with back, title, subtitle, avatar
- Scrollable message list (user, agent, tool cards)
- Message composer with actions, status row, and autocomplete overlay

## Inbox feed
- List of friend requests / notifications
- Optional floating action for “add friend” on tablet

## Settings panel
- Profile or logo header card
- Grouped list sections (accounts, connections, preferences)
- Embedded buttons for subscription and linking devices

## Permission prompt panel
- Inline decision list (allow, deny, allow for session)
- Visual left-border highlight for selected state

## Command palette (web / desktop)
- Centered modal with search field and results list
- Large shadow and blurred backdrop

## Artifacts
- List view of saved items
- Detail view with metadata and content blocks
- Edit view with input fields and action buttons

## Terminal connection sheet
- Explanation block + connection details + confirm/deny actions

## Usage & analytics panels
- Bars / charts with captions and totals

## Voice assistant status bar
- Compact status header with waveform/indicator
