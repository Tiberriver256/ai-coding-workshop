# Templates

Reusable page layout patterns.

## Auth / Welcome template
- Centered logo, title, subtitle
- Two stacked call-to-action buttons
- Portrait: vertically centered; landscape: two-column (logo + text/actions)

## Standard list page
- Top header (title + optional action)
- Scrollable content
- Grouped list sections with headers and footers

## Detail view template
- Header with back action
- Scrollable content area
- Optional sticky footer (composer or action bar)

## Chat template
- Header with session title + status
- Conversation list
- Composer docked to bottom

## Split view (tablet)
- Left sidebar (navigation, status)
- Right content panel (list or detail)
- Max width constraints for readability

## Modal / overlay template
- Fullscreen backdrop
- Centered card with rounded corners
- Primary and secondary actions

## Fullscreen confirmation template
- Icon + title + body copy
- Details list
- Action buttons stacked

## Form template
- Title + body text
- Inputs grouped by section
- Primary action at end
