# Atoms

Basic UI elements observed in the Happy mobile app.

## Buttons
- Primary pill button
  - Filled, high-contrast (dark fill, light text)
  - Sizes: small (24h), normal (32h), large (48h)
  - States: default, pressed (slightly darker), disabled (reduced opacity), loading (spinner)
- Secondary pill button
  - Text-only / inverted appearance
  - Used for secondary actions and cancellations
- Icon button
  - Square or circular tap target (32-44)
  - Used in headers, toolbars, and list rows
- Floating action button (FAB)
  - Circular 56, elevated shadow, plus icon
  - Appears in lower-right or floating context
- Wide FAB
  - Rounded rectangle with icon + label
  - Used for prominent “Create / New” actions

## Icons & pictograms
- Line icons for navigation and actions (24-29 size range)
- Badge icons for status (check, warning, error)
- Pictogram-style “brutalist” tiles for tab icons and optional nav accents

## Typography atoms
- Display / hero text (28-34)
- Title text (20-24)
- Section title / header (14-17, semibold)
- Body (14-16)
- Caption / meta (11-13)
- Monospace (12-14) for code, keys, and system content

## Inputs
- Text field (single-line)
  - Neutral background, rounded corners, subtle border
  - Placeholder in secondary text color
- Multiline text area
  - Taller, monospace option for keys/technical content
- Search input (inline text field with optional leading icon)
- Inline input with trailing action button (send)

## Toggles & selectors
- Switch (on/off)
- Radio button (dot-in-circle)
- Segmented selector / single-choice list (radio rows)

## Indicators
- Status dot (solid or pulsing)
- Badge (count pill, red background, white text)
- Dot indicator (tiny 6px dot)
- Activity spinner (small/large)
- Progress bars (usage / capacity)

## Avatars
- Circular avatar (generated or photo)
- Optional “flavor” badge overlay (small circular icon)
- Square avatar variant (for special contexts)

## Dividers & borders
- Hairline separators between list rows
- Section outlines for cards and grouped lists
