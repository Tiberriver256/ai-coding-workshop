# Molecules

Combinations of atoms used repeatedly across the product.

## Form and input groups
- Labeled input: label + field + helper/error text.
- Inline input row: field + action button (send/submit).
- Search bar: search icon + input + optional filter chip.
- Path picker row: path field + browse action + recent indicator.
- Token input: mono field + copy button + reveal toggle.

## Status and identity
- Agent status chip: status dot + label + elapsed time.
- Device row: device avatar + name + connection status + actions.
- User row: avatar + name + role + permissions badge.

## Task and board elements
- Task card header: title + status icons + assignee avatar.
- Task metadata row: tags + repo + last update.
- Column header: status dot + title + count + add action.
- Drag handle strip: subtle grip + hover highlight.

## Review and approval
- Diff file row: file name + change count + status indicator.
- Comment chip: avatar + line number + comment count.
- Approval decision row: approve/deny buttons + summary.

## Logs and activity
- Log line row: timestamp + source + message.
- Activity item: icon + action + actor + time.
- Alert banner: icon + message + action.

## Actions and utilities
- Action group: primary + secondary buttons aligned.
- Toolbar cluster: icon buttons + separators.
- Breadcrumb row: project > task > attempt.
