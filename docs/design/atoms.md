# Atoms

Smallest UI elements that express brand, status, and interaction feedback.

## Buttons
- Primary button: filled brand primary, used for create/start/approve.
- Secondary button: neutral surface with border for cancel/secondary actions.
- Tertiary button: text-only for low-priority actions.
- Destructive button: error-toned for irreversible actions (delete, stop attempt).
- Icon button: square or circular target for toolbar and card actions.
- Split button: primary action + dropdown for alternate execution paths.

States: default, hover, pressed, focus, disabled, loading (spinner).

## Links
- Inline link: text with underline on hover/focus.
- External link indicator: small icon appended to label.

## Typography atoms
- Display, page title, section title, body, caption, mono.
- Code token: mono with subtle background for inline identifiers.

## Icons
- Navigation icons: board, sessions, devices, settings.
- Action icons: create, start, stop, approve, reject, comment.
- Status icons: pending, running, waiting, success, failed.
- File type icons: code, config, docs.

## Inputs
- Text field (single-line).
- Text area (multi-line, optional mono for prompts).
- Search field (leading icon + clear control).
- Select dropdown (single choice).
- Combo input (editable + suggestions).
- Key input (masked or mono for tokens).

## Toggles & selectors
- Switch (on/off).
- Checkbox.
- Radio.
- Segmented control (view modes: board/list/diff).
- Stepper (incremental values like retry limits).

## Badges & pills
- Status badge (colored, icon + label).
- Count badge (numeric, compact).
- Tag pill (neutral or colored for labels).
- Permission badge (read-only, scoped, elevated, admin).

## Indicators
- Status dot (static or pulsing).
- Progress bar (determinate/indeterminate).
- Spinner (small/medium/large).
- Connection indicator (connected/connecting/offline).

## Avatars
- User avatar (circle).
- Agent avatar (distinct silhouette or glyph).
- Device avatar (monitor/phone icon container).

## Dividers & surfaces
- Hairline divider.
- Card surface.
- Panel surface.

## Feedback primitives
- Tooltip.
- Toast.
- Skeleton line/block.
